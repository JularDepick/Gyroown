## [0.1.2] - 2026-05-26

### Added
- **GUI 骨架**：完整的文件管理器式界面布局
  - 自定义标题栏（搜索框、主题切换、窗口控制）
  - 工具栏（新建文件夹、导入、导出、移动、删除、设置）
  - 侧边栏（虚拟目录树导航）
  - 文件列表（Details/Icons/Tiles 三视图切换）
  - 状态栏（选中计数、仓库路径、加密状态）
- **密码系统**：四种解锁方式
  - 6 位数字 PIN
  - 九宫格手势图案
  - 自定义密码（6-32 位字符）
  - 图片密码（点击坐标验证）
- **加密引擎**：RSA 2048 + AES-GCM 混合加密
  - PBKDF2 密钥派生（100,000 次迭代）
  - 非对称密钥对加密（私钥加密/公钥解密）
  - 分块存储支持（可配置块大小）
  - 安全擦除（覆写随机数据）
- **文件管理**：完整的 CRUD 操作
  - 文件导入（拖入/选择文件）
  - 文件导出（拖出/选择目标）
  - 文件删除（安全擦除）
  - 文件重命名
  - 文件夹创建/删除
  - 虚拟目录树
- **主题系统**：个性化设置
  - 跟随系统/浅色/深色主题
  - 8 种强调色预设
  - 中英文语言切换
  - 配置持久化
- **拖放功能**：拖入加密存储
- **后台驻留**：关闭窗口最小化到托盘
- **单实例检测**：防止重复启动
- **完整性检查**：启动时检查数据一致性
- **预览图生成**：图片文件自动生成缩略图
- **搜索功能**：文件名搜索 + 搜索历史
- **Banner 动画**：成功/错误提示滑入滑出效果
- **进度显示**：文件导入导出进度条
- **错误日志**：错误信息记录和查看
- **文件查看器**：图片/视频/音频/文本预览

## 版本说明

### 版本号规则
- 主版本号.次版本号.修订号
- 主版本号：重大功能变更或不兼容更新
- 次版本号：新功能添加，向后兼容
- 修订号：Bug 修复，向后兼容

### 发布流程
1. 更新版本号（Gyroown.csproj）
2. 更新 CHANGELOG.md
3. 执行 `dotnet build` 验证构建
4. 执行 `dotnet publish` 生成发布包
5. 创建 Git tag
6. 发布到 GitHub Releases
