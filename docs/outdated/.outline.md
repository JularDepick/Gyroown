# Gyroown 开发大纲

> 只列纲要不写细节。细节见 `v0.1.1-20260526.md`。

## 第 1 期：外壳 & 接口

- 应用生命周期
  - 单实例检测 (`AppInstance`)
  - 托盘驻留 (`H.NotifyIcon`)
  - 启动路由（首次 → 密码设置 / 已有 → 解锁）
- 主窗口布局
  - 自定义标题栏（拖拽区域 + P/Invoke 按钮）
  - 工具栏（CommandBar）
  - 虚拟目录侧边栏（TreeView）
  - 内容区（视图切换 + 文件列表桩）
  - 状态栏（选中计数 / 路径 / 加密状态）
- 密钥架构：用户密码 → userKey → 解密密钥对 → 私钥加密 / 公钥解密
- 服务接口定义
  - `IPasswordService` — 身份认证
  - `IEncryptionService` — 两层密钥 + 文件加解密
  - `IVaultService` — 仓库 CRUD（持有 vaultKey）
  - `IThemeService` / `IDragDropService`
  - 全部空桩实现
- 基础设施
  - 本地路径 `%USERPROFILE%\.Gyroown\`（`auth\` + `data\` + `key\`）
  - 异常处理策略
  - MicaBackdrop 降级检测

## 第 2 期：密码系统

- 密码控件接口 `IPasswordControl`
- 四种控件
  - PIN：6 位单字符 PasswordBox
  - 手势：3×3 Canvas + Polyline 连线
  - 自定义：PasswordBox + 显隐切换 + 字符校验
  - 图片：Uniform 缩放 + 比例坐标 + 序号圆标记
- UnlockWindow
  - 密码类型识别
  - 错误计数 + 锁定倒计时
- PasswordSetupWindow
  - 首次设置向导
  - 生成 vaultKey → 派生 userKey → 加密储存 vaultKey → 初始化仓库
- 密码策略模型

## 第 3 期：保密柜 UI

- VaultFileListView 控件
  - Details / Icons / Tiles 三视图
- 拖放
  - 拖入：Drop 事件 → 加密 → 刷新列表
  - 拖出：DragStarting 解密到 TEMP → StorageItems → 安全擦除
- 虚拟目录树绑定
- 右键上下文菜单
- 主题色切换
  - 跟随系统 / 浅色 / 深色
  - FrameworkElement.RequestedTheme
  - 持久化 settings.json
